# 官网
https://agentskills.io
https://github.com/agentskills/agentskills


### 特点：AgentSkills 获取静态指令规范。
	- 专业工作流
	- 捆绑资源（脚本、模板、参考资料）
	- 渐进式信息披露
	- 静态、可复用模式

### 拓展：MCP（模型上下文协议）连接外部工具和数据源的协议，获取动态资源数据。
	- 数据库连接
	- API 集成
	- 实时数据获取
	- 外部服务集成

### 关系：
	AgentSkills 和 MCP 互补，用 AgentSkills 获取知识，用 MCP 扩展功能。


# 目录规范
my-skill/
├── SKILL.md          # Required: instructions + metadata
├── scripts/          # Optional: executable code
├── references/       # Optional: documentation
└── assets/           # Optional: templates, resources


# 内容规范
SKILL.md
```
---
name: skill-name
description: A description of what this skill does and when to use it.
---
```



# Skills 加载器：openskills
-- OpenSkills 将 Anthropic 的技能系统 引入到所有 AI 编码代理中（Claude Code、Cursor、Windsurf、Aider）

### 工具地址
https://github.com/numman-ali/openskills

### 安装工具
npm i -g openskills

### 安装技能
Claude 示例技能：https://github.com/anthropics/skills

openskills install anthropics/skills [--global] [--universal]

openskills install anthropics/skills --global --universal
openskills install https://gitcode.com/GitHub_Trending/skills3/skills.git --global --universal

```
异常：Security error: Installation path outside target directory
解决：
① git clone https://gitcode.com/GitHub_Trending/skills3/skills.git 
② 复制：skills/* 到 .agent\skills/ 或 .claude/skills/
```

### 同步至 AGENTS.md，默认生成 AGENTS.md 文件
openskills sync


### 其他命令
openskills install <source> [options]  # Install from GitHub, local path, or private repo
openskills sync [-y] [-o <path>]       # Update AGENTS.md (or custom output)
openskills list                        # Show installed skills
openskills read <name>                 # Load skill (for agents)
openskills manage                      # Remove skills (interactive)
openskills remove <name>               # Remove specific skill



# Skills 加载器：add-skill
【！！！注：已更名为 skills: https://github.com/vercel-labs/skills】

### 工具地址
https://github.com/vercel-labs/add-skill


### 安装工具

npm install -g add-skill
add-skill xxx-skills
或
npx add-skill xxx-skills


### 其他命令

-g, --global 				# Install to user directory instead of project
-a, --agent <agents...> 	# Target specific agents (e.g., claude-code, codex). See Available Agents
-s, --skill <skills...> 	# Install specific skills by name
-l, --list 					# List available skills without installing
-y, --yes 					# Skip all confirmation prompts
-V, --version 				# Show version number
-h, --help 					# Show help



# Skills 加载器 skills：

### 工具地址
https://github.com/vercel-labs/skills


### 安装工具

npm install -g skills
skills add xxx-skills
或
npx skills add xxx-skills


### 命令操作
Usage: skills <command> [options]


其他命令：
npx skills list 			# List installed skills (alias: ls)
npx skills find [query] 	# Search for skills interactively or by keyword
npx skills remove [skills] 	# Remove installed skills from agents
npx skills update [skills] 	# Update installed skills to latest versions
npx skills init [name] 		# Create a new SKILL.md template


`add` 命令 Options：
-g, --global 				# Install to user directory instead of project
-a, --agent <agents...> 	# Target specific agents (e.g., claude-code, codex). See Available Agents
-s, --skill <skills...> 	# Install specific skills by name (use '*' for all skills)
-l, --list 					# List available skills without installing
-y, --yes 					# Skip all confirmation prompts
--copy 						# Copy files instead of symlinking to agent directories
--all 						# Install all skills to all agents without prompts

更多请使用 skills -h 查看。
