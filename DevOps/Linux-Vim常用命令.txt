Linux-Vim常用命令：

1.编辑模式：
	① 按「i」进入编辑模式，从光标当前位置开始输入；
	② 按「a」进入编辑模式，从光标当前位置下一个位置开始输入；
	③ 按「o」进入编辑模式，从下一行行首开始输入。

2.从编辑模式切换为命令模式按「ESC」键。

3.命令模式：
	① 撤销：u ；撤销后重做：Ctrl + r ；
	② 光标跳转到文件第一行：gg ；光标跳转到文件最后一行：Shift + g ；
	③ 光标跳转到行首：Home ； 光标跳转到行尾：End ；
	④ 删除光标当前行：dd ；
	⑤ 查找：/ + 查找关键词 + Enter ； 下一个：n ； 上一个：Shift + n ；
	⑥ 退出Vim编辑器：
		: w filename （输入 「w filename」将内容以指定的文件名filename保存）。
		: wq (输入「wq」，存盘并退出vi)。
		: q! (输入q!， 不存盘强制退出vi)。



拓展：

	① vim 列块选择模式(用途：快速添加、删除配置文件中的注释`#`):
		a.命令模式下， Ctrl + v 进入列块选择模式(VISUAL BLOCK)，移动光标进行列块选择。
		b.块删除：选择列块后，按 d 或者 Delete。
		c.块增加：选择列块后，按 Shift + i，编辑增加内容后，按 Esc 退出即可。
	② 显示行号：命令模式下输入 set number(nonumber) 显示(不显示)行号。
	③ 语法高亮：命令模式下输入 syntax on(off) 语法高亮开启(关闭)。
