Windows 软件默认安装到 C盘 的目录迁移方案


# Windows 创建链接 `mklink` 命令
mklink [参数] <链接目录> <源目录>

参数 `/D`、`\J` 都支持目录链接:
	/J : 移动 链接目录，相当于把源目录移动，源目录为空，会新建移动前的原链接映射关系。做目录链接更合适，兼容性好，推荐。
	/D : 移动 链接目录，源目录不变，链接目录被移动到新的位置，链接映射关系更新。支持 跨磁盘分区 或 网络共享目录创建链接。


# 迁移步骤：
① 关闭软件，移动目录 到 新位置。
② 使用 `mklink` 创建目录的快捷目录。


# 示例：
mklink /J "C:\Program Files\Docker" "D:\Program Files\Docker"


注：
① 需以管理员身份运行 `mklink` 命令。
② 建议使用绝对路径来创建链接。
③ 删除链接目录是安全的，不会删除源目录 及 源目录里面的文件。
