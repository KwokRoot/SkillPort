Windows Bat 脚本

# 示例

```bat
@echo off
chcp 65001 >nul
REM 这是一个注释
echo Hello, World!
pause
```

# 示例说明

@echo off
    关闭命令回显，避免在运行时显示每一条执行的命令。
    开头的 @ 表示连这一行也不回显。

chcp 65001 >nul
	将命令提示符窗口的字符编码设置为UTF-8，以确保支持中文或其他多字节字符。Windows默认的编码是 ASCII 编码。chcp 是改变控制台代码页的命令，65001 是 UTF-8 编码的代码页，nul 是将 chcp 命令的输出隐藏，不显示在控制台。

REM 这是一个注释
    注释行，不会被执行。也可以用 :: 替代。

echo Hello, World!
    输出文本。由于前面设置了 UTF-8，这里即使写中文也没问题。

pause
    暂停脚本，显示“请按任意键继续...”，防止窗口一闪而过（常用于双击运行 .bat 文件时）。
