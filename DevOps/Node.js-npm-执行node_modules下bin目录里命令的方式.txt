# npm 项目执行 `node_modules/.bin` 目录里命令的方式


① package.json scripts 中定义脚本
```
{
  "scripts": {
    "xxx": "node src/index.js"        // 直接写命令名，npm会自动在.node_modules/.bin中查找 xxx
  }
}
```
执行：
npm run xxx

② 使用 npx 命令临时执行，npx 会自动在当前项目的 node_modules/.bin中寻找命令，如果本地没有安装，它还可以临时下载并运行，用完即弃。
npx xxx

③ ./node_modules/.bin/xxx 直接执行


## 拓展

npx 是 npm v5.2.0 之后引入的命令行工具，用来执行包中的命令行工具。无需全局安装，直接运行本地 `node_modules/.bin` 中的命令，例 `npx xxx-server`。npm v7.0.0+ 后建议使用 `npm exec -- xxx-server`。
