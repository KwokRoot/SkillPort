Node.js-npm-安装环境配置


# 安装
https://nodejs.org/zh-cn/download


# 版本查看
node -v
npm -v


# 配置

### 配置 npm 缓存目录，全局模块目录

npm config set prefix "D:\kwok\data\node_global"
npm config set cache "D:\kwok\data\node_cache"


### 配置环境变量

PATH
	D:\kwok\soft\nodejs\
	D:\kwok\data\node_global
NODE_PATH
	D:\kwok\data\node_global\node_modules


### 配置 npm 仓库地址
npm config set registry https://registry.npm.taobao.org/


### 查看 npm 配置
① 查看单个配置
npm get registry
或 npm config get registry
② 查看 npm 配置
npm config ls
③ 查看 npm 配置(包含默认)
npm config ls -l
