# PyTorch GPU 版本安装
如果运行 `print(torch.__version__)` 结果为 `2.6.0+cpu`，则是因为 torch 版本不正确。正常应该为 `2.7.0+cu128`。

卸载：
pip3 uninstall torch torchvision torchaudio

根据 CUDA 版本重新安装(https://pytorch.org/get-started/locally/)：
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu129


> python
import torch
torch.cuda.is_available()

如果运行 `torch.cuda.is_available()` 结果为 True，则安装 PyTorch GPU 版本成功。
